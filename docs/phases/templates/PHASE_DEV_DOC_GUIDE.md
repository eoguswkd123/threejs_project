## 2. Version Guidelines (버전 가이드라인)

버전 형식: `MAJOR.MINOR.PATCH` (시간 기반 버전 관리)

| 변경 시점           | 버전 증가                  | 예시                  |
| ------------------- | -------------------------- | --------------------- |
| **같은 달 내 수정** | PATCH +1                   | 0.0.0 → 0.0.1 → 0.0.2 |
| **월이 바뀌면**     | MINOR +1, PATCH 리셋       | 0.0.5 → 0.1.0         |
| **연이 바뀌면**     | MAJOR +1, MINOR/PATCH 리셋 | 0.3.2 → 1.0.0         |

**예시 타임라인:**

```
2025-12-01  0.0.0  초기 문서 작성
2025-12-02  0.0.1  수정
2025-12-15  0.0.2  수정
2026-01-05  0.1.0  월 변경 (1월)
2026-01-10  0.1.1  수정
2027-01-01  1.0.0  연 변경
```

## 3. Checklist Guidelines (체크리스트 가이드라인)

| 상태   | 표기     | 설명               |
| ------ | -------- | ------------------ |
| 미완료 | `- [ ]`  | 아직 시작하지 않음 |
| 완료   | `- [✅]` | 구현 완료          |

**작성 예시:**

```markdown
### 3.1 Phase X.Y.1: [마일스톤]

- [✅] 항목 1 완료
- [✅] 항목 2 완료
- [ ] 항목 3 진행 중
```

## 4. Status Icons (상태 아이콘)

### Phase/Milestone 상태

| 아이콘 | 상태   | 설명                         |
| ------ | ------ | ---------------------------- |
| ✅     | 완료   | 100% 완료 및 검증됨          |
| 🔄     | 진행중 | 현재 작업 중 (퍼센티지 표시) |
| 📋     | 계획됨 | 설계 완료, 실행 대기         |
| ⏳     | 차단됨 | 의존성 미충족으로 대기       |
| ⏸️     | 보류   | 향후 재구현 예정             |

### 검증 체크리스트 상태

| 아이콘  | 의미      |
| ------- | --------- |
| ✅      | 검증 완료 |
| ⬜      | 미검증    |
| 📋 TODO | 예정됨    |

## 5. Naming Conventions (네이밍 규칙)

### docs/phases 폴더명

| 유형       | 규칙            | 예시                                |
| ---------- | --------------- | ----------------------------------- |
| Phase 폴더 | `##-PascalCase` | `01-Foundation/`, `02-CadFeatures/` |
| 기타 폴더  | lowercase       | `templates/`                        |

### docs/phases 파일명

| 유형        | 규칙                             | 예시                                     |
| ----------- | -------------------------------- | ---------------------------------------- |
| Phase 문서  | `[Phase번호]_SCREAMING_SNAKE.md` | `1.5_TEAPOT_DEMO.md`, `2A_DXF_VIEWER.md` |
| 가이드 문서 | `SCREAMING_SNAKE.md`             | `GUIDE.md`, `PHASE_DEV_DOC_GUIDE.md`     |
| 템플릿      | `SCREAMING_SNAKE.md`             | `PHASE_DEV_TEMPLATE.md`                  |
